# API

## Install below

### httpd24

    $ yum install -y httpd24*

### php71

    $ yum install -y php71*

### memacached

    $ yum install -y memcached

### git

    $ yum install -y git

### openssl-devel

    $ yum install -y openssl-*

### php-ed25519-ext

    $ git clone git://github.com/encedo/php-ed25519-ext.git
    $ cd php-ed25519-ext
    $ phpize
    $ ./configure
    $ make
    $ sudo make install
